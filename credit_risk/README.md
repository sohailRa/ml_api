# Credit Risk
A simple credit default risk predictor using Flask API

###Dataset
Kaggle's "Give Me Some Credit" for credit score default prediction

### Usage
1. python model.py (train and pickle a simple logistic regresson model)
2. python app.py
3.  python request.py
